Command reference

In the meantime there are lots of GUI commands for Bitrate Viewer that can be executed with keyboard and or mouse actions. They are more or less covered within all web pages. Here is a complete overview that may help you as a handy shortcut for your desktop:

Mouse driven activities:
 

Keyboard modifiers Mouse action Functionality Requirements
Activities inside the viewer window
- none - Drag & Drop Drop the selected file to open it instantly Drag a video file from Windows Explorer over the viewer window
Shift RightClick Toggle usage of gradient colors for view and caption bars - none -
- none - MouseMove Display small tick marks at every 5% of the view's width Position at the top of the view with an analyzed file
- none - MouseMove Track the current bitrate sample position Analyzed file displayed in the view
- none - LeftDoubleClick Toggle auto-scaling of the viewer window s.a.
Ctrl LeftClick Toggle average bitrate value marker line s.a.
Shift LeftClick Toggle maximum bitrate value marker line s.a.
Alt LeftClick Toggle minimum bitrate value marker line s.a.
Ctrl+Alt+Shift LeftClick Toggle all three bitrate value marker lines s.a.
Ctrl RightClick Toggle tracking of the mouse position in the view s.a.
Alt RightClick Toggle display of the mouse cursor tracking line s.a.
Ctrl+Alt+Shift RightClick Enter zoom at 1 sample/display unit at current position s.a.
- none - LeftBtnDown & Drag Set a zoomable area for parts of the view s.a.
- none - RightClick Enter zoomed view for the marked area Mouse position is within marked zoomable area
- none - LeftClick Cancel and erase marked zoomable area Any position, zoomable area is shown on screen
Ctrl RightBtnDown & Drag Point into the zoomable area and move it left or right s.a.
Shift RightBtnDown & Drag Resize the zoomable area to any new size s.a.
- none - LeftBtnDown & Drag Extended zoom increasing the zoom scale Display is already in zoomable state
Alt+Shift RightBtnDown Zoom out (-200%) at current position, decrease scale s.a.
Ctrl+Shift RightBtnDown Zoom in (+50%) at current position, increase scale s.a.
Alt+Shift LeftBtnDown Start continous automatic zoom-out, decrease scale s.a.
- none - WheelPushUp Zoom out from the display like with continous auto-zoom s.a.
Ctrl+Shift LeftBtnDown Start continous automatic zoom-in, increase scale Standard display view or any zoomable view
- none - WheelPullDown Zoom into the display like with continous auto-zoom s.a.
- none - MouseMove Display status window for first/last GOP's Current mouse position is on the left top sensitive area
- none - MouseMove Display status window for frame estimation and time elapsed Current mouse position is on the right top sensitive area
Ctrl RightClick Open context menu Status text window is shown on screen
Activities outside the viewer window
- none - LeftClick Exit from zoom mode, display standard view Click on zoom flag while zoomed view is active
- none - RightClick Toggle zoom flag shown as verbose/non-verbose s.a.
- none - LeftClick Move viewport of the zoomed view left or right Click is in zoom indicator beneath the bulls-eye while display is in zoomed state
Ctrl LeftBtnDown & Drag Move viewport of the zoomed view at any place horizontally along the view Button down is at any position on the zoom indicator during zoomed state
- none - LeftDoubleClick Exit from zoom mode, display standard view Click is on the zoom indicator's bulls-eye during zoomed state
- none - LeftClick Switch to seconds based bitrate calculation mode Click on the calculation method indicator with a valid file loaded and displayed on screen
- none - RightClick Switch to single frame based bitrate calculation mode s.a.
Ctrl LeftClick Switch to enhanced GOP bitrate calculation mode s.a.
Ctrl RightClick Switch to standard GOP based bitrate calculation mode s.a.
Alt LeftClick Select next bitrate calculation mode (rotate CW: S, G, X, F) s.a.
Alt RightClick Select previous bitrate calculation mode (rotate CCW: F, X, G, S) s.a.
- none - LeftClick Locate previous I-frame and position the cursor on it In frame analysis method click on the frame type indicator (I, P, B) with a valid file loaded and displayed on screen
- none - RightClick Locate next I-frame and position the cursor on it s.a.
Ctrl LeftClick Locate previous I-frame and center the view and place cursor on it s.a.
Ctrl RightClick Locate next I-frame and center the view and place cursor on it s.a.
- none - LeftClick Move the cursor one second back from its current position Click on the right hand panel's current time display with a file loaded
- none - RightClick Advance the cursor one second from its current position s.a.
Ctrl LeftClick Move the cursor one minute back from its current position s.a.
Ctrl RightClick Advance the cursor one minute from its current position s.a.
- none - LeftClick Stop toggling stream count and program/PID status for standard view Click on the left hand panel's stream count or program/PID status with a MPEG 2 TS file loaded
- none - RightClick Restart toggling stream count and program/PID status for standard view s.a.

Keyboard driven activities:
 

Keyboard shortcut Functionality
Ctrl-C Toggle usage of gradient colors for view and caption bars
Ctrl-A Toggle auto-scaling of the viewer window
Ctrl-Z Toggle zoom flag shown as verbose/non-verbose
C Close and exit the application
L Open file selection dialog to load a file
P Pause calculation/analysis of file during read
S Stop currently running read and analysis operation
F1 Open HTML help page (offline) in external browser
Alt-D Switch/toggle between double size mode and standard view
Alt-A Show about box, credits, etc.
F5 Recalculate current file, read from cache if data valid
Ctrl-F5 Force recalculation of current file from disk data
Ctrl-S Select second based bitrate calculation method
Ctrl-G Select standard GOP based bitrate calculation method
Ctrl-X Select enhanced GOP based bitrate calculation method
Ctrl-F Select single frame based bitrate calculation method
Ctrl-D Toggle drawing of plain or elliptic caption headers
Ctrl-I Open interface settings dialog
Ctrl-O Open properties dialog
Ctrl-U Force a manual online version check now
Ctrl-T Toggle display of dynamic tooltips (enable/disable)
